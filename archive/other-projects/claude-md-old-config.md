# Claude Code Configuration for Brainstorming Repository

This repository contains research and planning for mission-driven open-source projects.

## Project Context

**Purpose:** Research, ideate, and plan open-source alternatives to expensive SaaS products for Product Managers and AI Enthusiasts.

**Current Projects:**
1. **Claude Mail** - AI-powered B2B email campaign orchestration (research complete)
2. **8+ New Project Ideas** - See MISSION_DRIVEN_PROJECTS.md

## Available Custom Commands

Use these slash commands to work with project ideas:

- `/analyze-project` - Deep dive analysis of a specific project idea
- `/customer-discovery` - Generate interview questions for customer research
- `/mvp-spec` - Create detailed MVP specification document
- `/market-research` - Conduct comprehensive market research
- `/compare-projects` - Compare multiple projects and recommend priorities

## Development Principles

1. **Mission-Driven:** Every project should democratize access to powerful tools
2. **Open Source:** MIT or Apache 2.0 licensing
3. **AI-Powered:** Leverage Claude Sonnet 4.5 for intelligence
4. **User-Owned:** Self-hosting options, no vendor lock-in
5. **Sustainable:** Clear path to monetization without sacrificing mission
